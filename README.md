# Clothing Brand Management System

## Project Overview
The Clothing Brand Management System is a Java-based Object-Oriented Programming (OOP) project developed to manage products, inventory, orders, payments, users, and reports for a clothing brand. The system demonstrates the implementation of core OOP concepts such as encapsulation, inheritance, polymorphism, abstraction, interfaces, exception handling, and package management.

# Problem Statement
Managing a clothing brand manually can lead to inventory errors, difficulty in tracking orders, payment handling issues, and poor management of customer data. This project provides a computerized solution that automates clothing brand operations including product management, inventory control, order processing, payment methods, and reporting.

The system helps:
- Manage clothing products efficiently
- Track inventory stock
- Process customer orders
- Handle different payment methods
- Generate reports
- Improve overall management and organization

# Core Modules

## 1. Product Management

# OOP Features Implemented

## Encapsulation
Private data members with public getters/setters are used to protect data.

## Inheritance
Classes like Shirt, Hoodie, and Trouser inherit from the Product class.

## Polymorphism
Different payment methods override payment behavior.

## Abstraction
Abstract classes and interfaces are used for common system functionality.

## Interfaces
Interfaces such as:
- Searchable
- Trackable

## Exception Handling
Custom exceptions implemented:
- OutOfStockException
- InvalidLoginException
- InvalidPaymentException

## Package Management
The project is organized into packages:
- inventory
- orders
- payments
- products
- reports
- users

# How to Run the Project Using CMD

## Step 1: Extract the ZIP File
Extract the project folder.

## Step 2: Open CMD
Open Command Prompt and navigate to the project folder.

Example:
```bash
cd path_to_project_folder
```

## Step 3: Compile the Project
```bash
javac ClothingBrandSystem.java
```

## Step 4: Run the Project
```bash
java ClothingBrandSystem
```
